TikTok anunció en TikTok World 2026 el lanzamiento del TikTok Ads MCP (Model Context Protocol) server — el primer MCP de ads de una plataforma major que permite que Claude de Anthropic y ChatGPT de OpenAI gestionen, optimicen y reporten campañas publicitarias directamente en TikTok Ads, sin que el usuario tenga que abrir la plataforma. Este es el cambio más disruptivo en la operación de paid social desde la automatización de bidding. El MCP server de TikTok no simplifica la gestión de la plataforma — cambia el rol del profesional: de operar la plataforma directamente a supervisar un agente que opera por él. Para agencias: el primer equipo que domine este modelo tiene una ventaja operativa significativa — los mismos profesionales pueden gestionar el doble de campañas en el mismo tiempo.
