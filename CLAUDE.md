# CLAUDE.md · El Pulso Publicitario

## Sobre el proyecto
Newsletter HTML diario sobre noticias de publicidad y marketing.
Publicado vía GitHub Pages en: https://lorengb37-gif.github.io/mi-periodico-digital/

## Estructura del archivo `index.html`
- Exactamente **1384 líneas**
- Tailwind CSS via CDN (no modificar)
- Fonts: Bricolage Grotesque + Fraunces (no modificar)
- JavaScript de filtros funcional (no modificar)
- 5 secciones principales: Medios, Creatividad, Planning, Data, Content & Social

## Reglas críticas para actualizaciones diarias

### ✅ DEBES:
- Mantener exactamente las 1384 líneas
- Conservar todo el CSS, JavaScript, clases Tailwind, IDs y data-attributes
- Solo cambiar el contenido textual dentro de cada noticia
- Buscar noticias reales con web_search (no inventar)
- Incluir links REALES a las fuentes originales
- Incrementar el número de edición (+1 cada día)
- Usar el formato editorial juvenil y profesional en español

### ❌ NO DEBES:
- Modificar `<head>`, `<style>`, scripts JS
- Inventar URLs, estadísticas o nombres de empresas
- Cambiar la estructura HTML de las cards
- Romper los filtros (data-filter, data-category)
- Agregar o eliminar secciones

## Secciones a actualizar cada día

1. **Ticker superior** — 7 frases flash con emojis
2. **Editorial** — Título + 2 párrafos de análisis
3. **Medios** — 3 noticias (1 principal + 2 secundarias)
4. **Creatividad** — 2 noticias (campañas, casos creativos)
5. **Planning** — 2 noticias (insights, movimientos de cuentas)
6. **Data** — 2 noticias (IA, privacy, métricas)
7. **Content & Social** — 1 noticia (plataformas, creators)
8. **Mood del día** — 4 cards con vibra del día
9. **Water cooler** — 4 frases para conversaciones

## Formato de cada noticia
- Chip con categoría
- Titular llamativo
- Bajada (resumen 2-3 líneas)
- "Por qué importa" (análisis estratégico)
- "Acción sugerida" (qué hacer mañana)
- Links a fuentes reales

## Topics relevantes a buscar
- Upfronts y eventos de la industria
- Campañas virales (Adidas, Coca-Cola, Nike, etc.)
- Plataformas: TikTok, Meta, Google, Amazon, Disney, Netflix
- IA aplicada a publicidad
- Movimientos de cuentas de agencias
- Cambios regulatorios (privacy, AI)
- Noticias del Mundial 2026 publicitarias
- Movimientos ejecutivos importantes
